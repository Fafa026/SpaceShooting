# Space Shooter
Tervetuloa Space Shooting peliin! Tämä on yksinkertainen avaruusammuspeli, joka on toteutettu käyttäen Unity 2D. Pelissä pelaaja ohjaa avaruusalusta ja yrittää selviytyä avaruudessa liikkuvista vihollisista ampumalla niitä.

    Space Shooter kiehtova ammuskelupeli, joka vie pelaajat jännittäville
    avaruustaisteluille. Peli tarjoaa intensiivisen pelikokemuksen, joka haastaa pelaajien
    taituruuden.

## Peli on toteutettu 2D-grafiikalla
    Mikä tarkoittaa, että pelaajat näkevät pelin ylhäältä katsottuna. Tämä näkökulma
    tarjoaa hyvän yleiskuvan pelimaailmasta.

## Voimat
    Voima-upgradejen kerääminen antaa pelaajille mahdollisuuden
    parantaa avaruusalustaan ja mukauttaa pelityyliään. Tämä luo strategisen elementin
    peliin.
    
## Ominaisuudet

- Intuitiivinen peliohjaus: Liiku avaruusaluksellasi nuolinäppäimillä ja ammu vihollisia hiiren vasemmasta napista.
- Viholliset: Viholliset liikkuvat satunnaisesti avaruudessa ja yrittävät tuhota pelaajan avaruusaluksen.
- Pistejärjestelmä: Pelaaja saa pisteitä jokaisesta tuhotusta vihollisesta. Tavoitteena on saavuttaa mahdollisimman korkea pistemäärä.
- Äänitehosteet: Pelissä on äänitehosteet, jotka lisäävät pelaamisen immersiota.
